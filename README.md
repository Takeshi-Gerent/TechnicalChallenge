# TechnicallChallenge

## Requisitos
* Docker
* Antes de executar o comando abaixo verifique se as portas 5001 e 5002 liberadas no docker.

## Executar aplicação
```
> git clone https://github.com/Takeshi-Gerent/TechnicallChallenge
> cd TechnicalChallenge
> docker-compose up
```

## Endereço das API's
* Api1: http://localhost:5001
* Api2: http://localhost:5002
